Code

Codewars kata

Given a lottery ticket (ticket), represented by an array of 2-value arrays, you must find out if you’ve won the jackpot. Example ticket: [ [ ‘ABC’, 65 ], [ ‘HGR’, 74 ], [ ‘BYHT’, 74 ] ]

If the character code of any of the characters in the string matches the number, you get a mini win. Note you can only have one mini win per sub array.

Once you have counted all of your mini wins, compare that number to the other input provided (win). If your total is more than or equal to (win), return ‘Winner!’. Else return ‘Loser!’.

Kata solution

function bingo(ticket, win) {
                        let counter = 0;
                        ticket.forEach((item) => {
                            let miniwin = 0;
                            for (let i = 0; i < item[0].length; i++) {
                                if (item[0].charCodeAt(i) === item[1]) { miniwin = 1 }
                            }
                            miniwin > 0 ? counter = counter + 1 : counter = counter;
                        });
                        return (counter >= win ? "Winner!" : "Loser!");
                    }
